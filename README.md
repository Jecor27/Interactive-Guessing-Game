# Interactive-Guessing-Game

for practice purposes by creating an interactive guessing game. This exercise reinforces the use of conditional execution (if/else statements) and loops (for/while) in a practical scenario.

  - Use loops to repeat actions based on user input.

  - Implement conditional statements to compare values and control the game flow.

  - Use JavaScript functions such as prompt() for user input and alert() or console.log() for output.

  - Handle basic input validation and provide appropriate feedback.
